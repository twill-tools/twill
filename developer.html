Developing twill¶

twill is entirely written in Python. You will need Python 3.8 or newer to use and develop it.

Package tests¶

twill comes with several unit tests. They depend on pytest and Flask. To run them, type ‘pytest’ in the top level directory. To run an individual test, you can use the command pytest tests/test_something.py.
